## Individual Directions

Hereâ€™s how you apply spacing to specific sides.

### Top (`.mt-{size}`)
```html
<div class="mt-16">Margin Top 16px</div>
```

### Right (`.mr-{size}`)
```html
<div class="mr-16">Margin right 16px</div>
```

### Bottom (`.mb-{size}`)
```html
<div class="mb-16">Margin Bottom 16px</div>
```

### Left (`.ml-{size}`)
```html
<div class="ml-16">Margin Left 16px</div>
```
